# Justin Narusis - Personal Portfolio

This is the repository for my personal portfolio website, showcasing my journey in IT, my projects, and professional experiences. It includes information on my skills, certifications, and portfolio of work, as well as a blog documenting my ongoing learning and career growth.

**Live Site**: [https://jnarusis.github.io](https://jnarusis.github.io)

## Website Overview

This website serves as a professional portfolio and CV, highlighting my expertise in IT, cloud computing, networking, and system administration.

### Main Sections:
- **Home**: A quick introduction to who I am and what I do.
- **About Me**: Detailed information about my background, skills, and career goals.
- **Portfolio**: A showcase of the projects I've worked on, including descriptions and links to code or live demos.
- **Learning Journey**: A blog documenting my continuous learning, certifications, and projects.
- **Contact**: A contact form to reach me for collaborations, job inquiries, or general networking.

## Technologies Used

This website is built using the following technologies:

- **HTML5** and **CSS3**: For structuring and styling the website.
- **JavaScript**: For dynamic content and functionality.
- **GitHub Pages**: The website is hosted using [GitHub Pages](https://pages.github.com/), a free hosting service by GitHub for static websites.
