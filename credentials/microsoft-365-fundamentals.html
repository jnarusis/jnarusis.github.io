Microsoft Certified: Fundamentals (MS-900)

Certification Overview

This certification validates essential knowledge of Microsoft 365 services, modern workplace architecture, and cloud computing principles. It provided a strong foundation in identity and access management, licensing models, security policies, and service-level integration across Microsoft's SaaS ecosystem.

Topics Covered

  • Cloud computing models: IaaS, PaaS, SaaS
  • Microsoft 365 platform architecture
  • Core services: Teams, SharePoint, Exchange, OneDrive
  • Identity, access, and security management
  • Compliance, licensing, and pricing structures
