AI Toolkit CLI – modular, menu-driven tool for generating and explaining scripts and concepts

Project Overview

This project is a lightweight and modular command-line assistant built with Python and OpenAI. It helps users generate scripts, break down Bash or PowerShell commands, and understand Linux/Windows concepts. Designed for both learning and practical use, it runs locally and supports both Bash and PowerShell environments.

Features

  • Generate Bash or PowerShell scripts from plain-language prompts
  • Explain Linux and DevOps concepts
  • Break down Bash or PowerShell scripts line by line
  • Explore key Windows administration topics via natural language
  • Clean, readable menu interface with future expansion in mind

Tools & Technologies

  • Python 3
  • OpenAI API (GPT-4o)
  • Command-line interface (Bash / PowerShell)
  • dotenv for API key management
  • Git and GitHub for version control

Skills Demonstrated

  • Python scripting and CLI development
  • API integration and environment variable management
  • Prompt design for AI tools
  • Cross-platform tool design (Windows + Linux)
  • Clean code organization and version tracking
